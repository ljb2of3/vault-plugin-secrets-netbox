# Security Policy

## Supported Versions

Only the latest release of `vault-plugin-secrets-netbox` receives security
updates. If you are running an older version, please upgrade before reporting.

## Reporting a Vulnerability

Please report security vulnerabilities **privately** — do not open a public
issue.

Use GitHub's [private vulnerability reporting][pvr] to submit a report:

1. Go to the [Security tab][security-tab] of this repository.
2. Click **Report a vulnerability**.
3. Fill out the advisory form with as much detail as you can (affected
   version, reproduction steps, and impact).

We will acknowledge your report, work with you on a fix, and credit you in the
resulting advisory unless you prefer to remain anonymous.

[pvr]: https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability
[security-tab]: https://github.com/ljb2of3/vault-plugin-secrets-netbox/security
